Requirements

  • Browser: Chrome, Edge, or Opera (Web Serial API required)
  • Device: LilyGO T-Deck Plus (ESP32-S3, 8MB Flash)
  • Cable: USB-C data cable (not charge-only)

How It Works

  • 1. Connect T-Deck via USB-C
  • 2. Click "Flash Firmware" and select the serial port
  • 3. Wait for the firmware to be written
Tip: By default, only the firmware is updated - settings and messages are preserved. Check "Full install" for a clean wipe.
